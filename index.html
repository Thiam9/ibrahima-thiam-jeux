Ce jeux est un projet que j'ai réalisé pour ma formation Full stack JS à l'ifocop de paris.
Le code est entiérement écrit en Javascript. Le but du jeux est simple, casse les briques pour faire apparaitre mes compétences cachées derriéres les briques.

Utilise les flèches du clavier pour bouger la raquette. Bonne chance et fais gaffe de tu n'a que 3 vies.
